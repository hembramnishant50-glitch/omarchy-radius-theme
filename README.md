# Omarchy-Radius

OS: Omarchy (Arch-based)

- Install
```bash
omarchy-theme-install https://github.com/hembramnishant50-glitch/omarchy-radius.git
   ```

## Note :- Fix Hyprlock
Copy hyprlock.conf  to /home/user_name/.config/hypr  - For Hyprlock    
Edit the Hyprlock.config  
1. Change Profile Image - change your path
2. Change User Label - search USER to your custom name
